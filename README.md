# Dota3
Dota3 gaming game for players
### 1. Projekta mērķis (Analīze)
Izveidot nākamās paaudzes MOBA spēli, kas apvieno klasisko Dota 2 pieredzi ar modernākajām tehnoloģijām, uzlabotu grafiku un taisnīgu spēlētāju atlases sistēmu. Projekta fokuss ir uz spēlētāju kopienas uzlabošanu un tehnisko progresu.

### 2. Galvenās prasības (Requirements)
* **Jauns grafiskais dzinējs:** Reālistisks apgaismojums un fizikas efekti.
* **Viedais matchmaking:** Sistēma, kas izolē "toksiskus" spēlētājus.
* **VR atbalsts:** Iespēja vērot spēles virtuālajā realitātē.
* **Dinamiska karte:** Dienas/nakts cikls, kas maina upju līmeni un ceļus.
* **Balss vadība:** Iespēja dot pamatkomandas varonim ar balsi.
* **Trenera loma:** Sestais spēlētājs komandā stratēģiskai vadībai.
* **Cross-platform:** Spēlēšana starp PC un konsolēm.
### 3. Tehnoloģijas un rīki (Tools)
* **Projektu vadība:** Trello (darbu plānošana un izpildes kontrole).
* **Versiju kontrole:** GitHub (koda un dokumentācijas glabāšana).
* **Metode:** Agile / Scrum (darbs divu nedēļu sprintos).
